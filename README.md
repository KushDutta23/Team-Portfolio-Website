# Team Innovate - Portfolio Website

Welcome! This is a multi-page portfolio website for a fictional team called "Team Innovate."

**This repository represents my very first web development project!** It was built from scratch as a way to learn and apply the fundamental concepts of HTML, CSS, and vanilla JavaScript. The goal was to create a clean, responsive, and functional website that introduces a team, outlines their skills, and provides a way to contact them.

## 🚀 Features

*   **Fully Responsive Design:** Adapts to all screen sizes, from mobile phones to desktop computers.
*   **Multi-Page Layout:** A clean structure with 5 distinct pages:
    *   Home
    *   About
    *   Skills
    *   Portfolio / Team
    *   Contact
*   **Team Member Showcase:** A dedicated section on the Portfolio page to introduce the team members using a responsive grid layout.
*   **Mobile-Friendly Navigation:** A functional "hamburger" menu for easy navigation on smaller devices.
*   **"Coming Soon" Section:** A placeholder for the team's future collective portfolio.

---

## 🛠️ Technologies Used

This project was built using the core technologies of the web:

*   **HTML5:** For the structure and content of the website.
*   **CSS3:** For all styling, layout, and responsiveness.
    *   **Flexbox & CSS Grid:** Used for creating modern, responsive layouts.
    *   **Google Fonts:** For custom typography (`Lato`, `Open Sans`, `Roboto`).
*   **Vanilla JavaScript:** For DOM manipulation to create the interactive mobile navigation menu.

## 🌱 Learning Outcomes

As my first project, this was an incredible learning experience. Key takeaways include:

*   **Semantic HTML:** Structuring a website logically for both browsers and accessibility.
*   **CSS Fundamentals:** A deep dive into the box model, positioning, and styling.
*   **Responsive Design:** Understanding and implementing media queries, flexible units, and mobile-first principles.
*   **Layout Techniques:** Gaining hands-on experience with both **Flexbox** for component alignment and **CSS Grid** for overall page layout.
*   **Basic DOM Manipulation:** Using JavaScript to add and remove CSS classes to make the website interactive.
*   **File and Folder Organization:** Learning how to structure a project for clarity and scalability.
